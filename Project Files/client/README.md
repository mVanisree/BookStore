# 📚 BookHive - E-commerce Frontend

## Description

BookHive is a React.js-based e-commerce frontend for browsing and purchasing books. The application demonstrates modern React concepts and state management using Redux Toolkit.

## Features

- Home Page
- Product Listing
- Product Details
- Shopping Cart
- Search Books
- Category Filter
- Quantity Update
- Remove from Cart
- Total Price Calculation
- Responsive Design

## Technologies Used

- React.js
- Redux Toolkit
- React Router DOM
- CSS
- Vite

## Installation

```bash
npm install
npm run dev
```
